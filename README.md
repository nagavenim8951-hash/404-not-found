# 404 Not Found Page

This project is a simple and responsive **404 Not Found** error page created using **HTML & CSS**.

It displays a message when a user lands on a page that doesn't exist,along with an illustration and a button to go back to the haomepage.

## DEMO
[DEMO](https://htmlpreview.github.io/?https://raw.githubusercontent.com/nagavenim8951-hash/404-not-found/refs/heads/main/index.html)

## Features

1.Clean and modren design.

2.simple layout using Flexbox.

3.Responsive for different screen sizes.

4.Contains an image,description and a button.

2.CSS
